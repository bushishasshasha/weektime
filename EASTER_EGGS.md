# 彩蛋触发规则

彩蛋会在切换任务完成状态后，通过右下角 toast 显示。当前规则按优先级从高到低触发：

1. **一周全清彩蛋**
   - 条件：完成某个任务后，当前导入文本里的所有任务都处于已完成状态。
   - 概率：100%。
   - 文案池：当前主题的 `weekClear`。

2. **当天全清彩蛋**
   - 条件：完成某个任务后，该任务所属星期的所有任务都处于已完成状态，但整周还没全清。
   - 概率：100%。
   - 文案池：当前主题的 `dayClear`。

3. **稀有彩蛋**
   - 条件：完成某个任务后，没有触发一周全清或当天全清。
   - 概率：7%。
   - 文案池：当前主题的 `rare`。

4. **分类完成彩蛋**
   - 条件：完成某个任务后，没有触发上面的特殊彩蛋。
   - 概率：剩余情况 100%。
   - 文案池：按任务类别选择，例如 `game`、`study`、`fitness`、`paper`、`work`；没有匹配类别时使用 `default`。

5. **撤回完成彩蛋**
   - 条件：把已完成任务切回未完成。
   - 概率：100%。
   - 文案池：当前主题的 `undo`。

## 当前主题文案风格

- **东雪莲主题**：偏“莲莲 / 爆杀 / 破防 / 狠活 / 班味 / 举办 / 0 帧连招”的混沌梗。
- **Taffy 主题**：偏“谢谢喵 / 蛋包饭魔法 / 急急急 / 0 帧起手 / 举办 / 复活甲 / 混沌电波”的混沌梗。
- **嘉然主题**：偏“嘉心糖 / 嘉然今天吃什么 / 关注嘉然 / 嘉油 / 小恶魔 / 草莓 / 拿下二楼 / 圣嘉然”的混沌偶像梗。
- **默认主题**：使用轻度发癫、偏鼓励的普通文案，不走皮肤主题的混沌梗。

## 当前数量

- **东雪莲主题**：60 条。
- **Taffy 主题**：60 条。
- **嘉然主题**：60 条。
- **默认主题**：12 条。

## 实现位置

- 彩蛋文案池：[data.js](./data.js) 里的 `themePacks.*.easterEggs`。
- 触发逻辑：[timetable.html](./timetable.html) 里的 `themeEasterEggMessage(item, completed, days)`。
- 当天/整周完成判断：[timetable.html](./timetable.html) 里的 `completionStats(days, day)`。

## 注意

当前稀有彩蛋概率写死为 `Math.random() < 0.07`，即 7%。如果之后要做可配置概率，可以把它挪进各主题的 `themePacks.*.easterEggs.rareChance`。
